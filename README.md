# Sistema Acadêmico

Projeto desenvolvido com React + Firebase (Auth + Firestore)

## Funcionalidades

- Edição e exclusão

## Tecnologias
- React (Vite)
- Firebase Authentication
- Firestore

## Como rodar
npm install
npm run dev
